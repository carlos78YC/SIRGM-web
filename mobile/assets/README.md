# Assets de la Aplicación

Esta carpeta debe contener los siguientes archivos de imagen:

## Archivos Requeridos

1. **icon.png** (1024x1024 px)
   - Icono de la aplicación
   - Debe ser cuadrado
   - Se usa en iOS y Android

2. **splash.png** (1284x2778 px recomendado)
   - Imagen de splash screen
   - Se muestra al iniciar la aplicación
   - Fondo blanco recomendado

3. **adaptive-icon.png** (1024x1024 px)
   - Icono adaptativo para Android
   - Se muestra en la pantalla de inicio de Android

4. **favicon.png** (48x48 px)
   - Favicon para la versión web
   - Opcional si no planeas usar la versión web

## Crear Assets

Puedes crear estos assets usando herramientas como:
- [Expo Asset Generator](https://www.favicon-generator.org/)
- [App Icon Generator](https://appicon.co/)
- Cualquier editor de imágenes (Photoshop, GIMP, Figma, etc.)

## Placeholder Temporal

Mientras creas los assets reales, puedes usar imágenes placeholder. La aplicación funcionará sin estos archivos, pero verás errores en la consola.

Para desarrollo rápido, puedes crear imágenes simples de 1024x1024 px con un color sólido y el texto "SIRGM".
